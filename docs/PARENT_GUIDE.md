# 👨‍👩‍👧‍👦 Посібник для батьків

**SmartLessa** — це освітня AR-гра. Ми дбаємо про безпеку вашої дитини.

### Чи безпечно це?
1. **Приватність:** Застосунок не "бачить" обличчя дитини у значенні ідентифікації. Він лише будує цифрову сітку для накладання віртуального вінка.
2. **Контроль:** Усі дані про квест зберігаються лише на вашому смартфоні.
3. **Фізична безпека:** Будь ласка, нагадайте дитині, що під час руху з телефоном треба дивитися під ноги та навколо себе.

### Як видалити дані дитини?
Натисніть та утримуйте кнопку 🔄 (оновлення) протягом 2 секунд, або просто очистіть кеш браузера.
