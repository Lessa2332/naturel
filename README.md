**SmartLessa** — це інтерактивний освітній квест з доповненою реальністю (WebAR) для учнів 6-х класів НУШ.

### 🚀 Особливості
- **Пошук:** Сканування QR-кодів рослин.
- **Нагорода:** Створення віртуального вінка з патріотичними стрічками (активується посмішкою).
- **Безпека:** 100% анонімність, локальна обробка даних.
